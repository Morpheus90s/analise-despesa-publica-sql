# Questão 3 — Visualizações

# Ferramenta utilizada
Power BI Desktop.

# Fonte dos dados
Os dados foram extraídos do banco PostgreSQL remoto via DBeaver e exportados em arquivos CSV, devido a limitações de conexão direta do Power BI com o banco remoto.

# Visualizações construídas

# 1. Comparação entre valores empenhados e liquidados
Gráfico de colunas comparando os valores totais empenhados e liquidados ao longo dos anos, permitindo identificar diferenças entre o planejamento do gasto público e sua efetiva execução.

# 2. Indicadores consolidados
Foram utilizados cartões (KPIs) para apresentar os valores totais empenhados e liquidados no período analisado.

# Observações técnicas
Os cartões apresentam o total consolidado do período, enquanto o gráfico detalha os valores por ano, respeitando o contexto de filtro aplicado.

As visualizações estão disponíveis no arquivo `visualizacoes/questao_3_visualizacoes.pdf`.
