# Questão 3 — Visualizações

O objetivo desta etapa foi criar visualizações simples e claras para comunicar os principais resultados obtidos na análise do fluxo da despesa pública, com foco na comparação entre os valores empenhados e liquidados.

# Ferramenta utilizada
Power BI Desktop.

## Fonte dos dados
Os dados foram extraídos do banco PostgreSQL disponibilizado no desafio, utilizando o DBeaver.  
Devido a limitações de conexão direta entre o Power BI e o banco remoto, os dados foram exportados em arquivos CSV para posterior utilização no Power BI.

# Visualizações construídas

## 1. Comparação entre valores empenhados e liquidados
Foi construído um gráfico de colunas comparando os valores totais empenhados e liquidados no período analisado.

Os dados disponíveis concentram-se no ano de 2024, permitindo observar a diferença entre o valor inicialmente empenhado e o valor efetivamente liquidado, auxiliando na compreensão do comportamento da execução da despesa pública.

## 2. Indicadores consolidados
Foram utilizados cartões (KPIs) para apresentar os valores totais empenhados e liquidados no período analisado.

Esses indicadores fornecem uma visão geral e resumida dos montantes envolvidos, facilitando a leitura e interpretação dos dados.

## Observações técnicas
- Os cartões exibem os valores totais consolidados do período analisado.
- O gráfico apresenta a comparação entre empenhado e liquidado conforme os dados disponíveis.
- As visualizações foram construídas com foco em clareza, simplicidade e leitura objetiva.
